# Changelog

Release notes live on **GitHub Releases**:
<https://github.com/Kuestenlogik/Bowire/releases>

Each tag (`v1.x.y` and the matching `-rc.N` pre-releases) carries a
dedicated release page with highlights, breaking changes (none so
far), and upgrade instructions. That page is the single source of
truth; this file is intentionally kept as a pointer so the workflow
doesn't have to maintain release notes in two places.

To see the notes for the version you have installed:

    bowire --version
    open https://github.com/Kuestenlogik/Bowire/releases/tag/v<that-version>
