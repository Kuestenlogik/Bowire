Install from a package manager

The fastest path for most developers — NuGet for embedded integrations, the .NET global tool for a standalone CLI, or your platform’s native package manager.

NuGet packages

Every plugin ships as a standalone NuGet package so you install only what you need. All published by Küstenlogik on nuget.org.

Core

  • Kuestenlogik.Bowire
    Host integration, plugin loader, discovery pipeline.
  • Kuestenlogik.Bowire.Tool
    Standalone bowire CLI + browser UI, installable as a global .NET tool.

Protocol plugins

  • Kuestenlogik.Bowire.Protocol.Grpc
    gRPC with Server Reflection, all four streaming types, HTTP transcoding.
  • Kuestenlogik.Bowire.Protocol.Rest
    REST / OpenAPI, AWS Sig v4, Swagger import.
  • Kuestenlogik.Bowire.Protocol.GraphQL
    GraphQL with schema introspection + subscriptions (WS / SSE).
  • Kuestenlogik.Bowire.Protocol.SignalR
    SignalR hubs with typed method invocation + real-time channels.
  • Kuestenlogik.Bowire.Protocol.WebSocket
    Raw WebSocket with text + binary frames, sub-protocol selection.
  • Kuestenlogik.Bowire.Protocol.Sse
    Server-Sent Events — event-stream discovery and inline rendering.
  • Kuestenlogik.Bowire.Protocol.Mcp
    Model Context Protocol adapter + client, any protocol as MCP tools.
  • Kuestenlogik.Bowire.Protocol.Mqtt
    MQTT 3.1.1 / 5.0 via MQTTnet — topics, QoS, retain, duplex channels.
  • Kuestenlogik.Bowire.Protocol.SocketIo
    Socket.IO v2/v3/v4 — namespaces, rooms, ack, binary events.
  • Kuestenlogik.Bowire.Protocol.OData
    OData CSDL discovery, typed filter / orderby / expand, batch requests.
  • Kuestenlogik.Bowire.Protocol.Storm
    Küstenlogik Storm broker — native protocol or Kafka-compat wire, in-process tap via storm://embedded.
  • Kuestenlogik.Bowire.Protocol.Kafka
    Apache Kafka via Confluent.Kafka — topics, mTLS + SASL, Avro decode via Schema Registry.
  • Kuestenlogik.Bowire.Protocol.Dis
    IEEE 1278 DIS — UDP multicast PDU stream, structured EntityState / Fire / Detonation rendering.
  • Kuestenlogik.Bowire.Protocol.Udp
    Raw UDP datagrams — UTF-8 detection, base64 + hex dump fallback for binary.

Third-party plugins

Bowire’s protocol system is open — anyone can ship a plugin against the IBowireProtocol interface and publish it to NuGet. Packages tagged bowire-plugin show up in the discovery feed below. These are community-maintained, not official Küstenlogik releases — audit the source before running them in production.

.NET global tool — standalone CLI + UI

Install once, point at any running server. No code changes to your host app.

$ dotnet tool install -g Kuestenlogik.Bowire.Tool
$ bowire --url https://your-server

Windows package managers Planned

Install the standalone tool through the package manager you already use — no .NET tool chain on the host required.

Native installers and portable bundles

Platform-native installers, xcopy-style self-contained bundles, or tagged releases from GitHub — whichever fits how you ship software.

Native installers Planned

Signed MSI for Windows, DEB for Debian / Ubuntu, RPM for Fedora / RHEL / SUSE. For shops that prefer a platform-native install path over the .NET tool chain.

  • bowire-x64.msi Windows 10 / 11, signed, per-user or machine-wide install.
  • bowire_amd64.deb Debian / Ubuntu, installs to /usr/bin/bowire via apt.
  • bowire.x86_64.rpm Fedora / RHEL / SUSE, installable via dnf or yum.
See the roadmap →

Self-contained / portable

Zip or tarball per platform with the .NET runtime bundled. Extract, run the bowire binary — no SDK, no installer, no registry entries, no system changes. Perfect for air-gapped networks, USB-stick installs, ephemeral CI runners, or anywhere you want an xcopy deployment.

  • bowire-win-x64.zip Windows x64 — unzip, run bowire.exe.
  • bowire-linux-x64.tar.gz Linux x64 — tar -xzf, run ./bowire/bowire.
  • bowire-linux-arm64.tar.gz Linux ARM64 — Raspberry Pi 4+, AWS Graviton, etc.
  • bowire-osx-arm64.tar.gz macOS Apple Silicon (M1+).
  • bowire-osx-x64.tar.gz macOS Intel.
