Beyond the workbench

Six subcommands you’d reach for outside the browser. Same binary as the workbench, real exit codes, fits in any pipeline.

bowire scan

Multi-protocol security scanner. Built-in passive checks plus the 8000+ projectdiscovery/nuclei-templates via --nuclei. Emits SARIF 2.1.0 for GitHub Code Scanning / GitLab / Azure DevOps.

bash 
bowire scan --target https://api.example.com \
            --nuclei ~/nuclei-templates \
            --out findings.sarif --severity medium

bowire mock

Replay any captured recording as a live mock endpoint — REST, gRPC (unary + streaming + bidi), WebSocket, SSE, SignalR, Socket.IO, MQTT, GraphQL subscriptions. Hot-reload, chaos injection, capture-on-miss.

bash 
bowire mock --recording session.bwr \
            --port 5099 --hot-reload

Mock server docs →

bowire fuzz

Schema-aware field-level fuzzer. Knows not to throw SQL-injection at a lat field; image.bytes gets magic-byte mutation, not XSS strings. Categories: sqli / xss / pathtrav / cmdinj with a baseline-diff oracle.

bash 
bowire fuzz --target https://api.example.com \
            --template recordings/order-create.bwf \
            --field '$.customer.email' --payloads xss

bowire jwt

Decode, tamper, re-sign JWT tokens from the terminal. The classic alg:none downgrade + claim-swap + weak-secret toolkit, scriptable enough for CI gates and security-regression suites.

bash 
bowire jwt tamper <token> \
                  --set isAdmin=true \
                  --secret weak123

bowire mcp serve

Expose Bowire itself as an MCP server. AI agents (Claude Desktop, Cursor, Copilot) drive discovery / invoke / record / mock through their conversation. Stdio for desktop wiring, HTTP for over-the-wire clients.

bash 
bowire mcp serve --bind stdio

MCP adapter docs →

bowire proxy

Intercepting forward proxy with HTTPS MITM (auto-generated CA, on-the-fly leaf-cert minting). Captured flows stream live into the workbench and convert with one click into Bowire recordings — ready to fuzz, mock, scan, or replay.

bash 
bowire proxy --port 8888 --api-port 8889
