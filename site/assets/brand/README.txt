Bowire - Brand assets
=====================

Logos (SVG, scalable). Use the *-dark files on light backgrounds and the
*-light files on dark backgrounds.

  bowire-logo-dark.svg   / -light.svg   Full logo (bowline + Circle-B). Primary.
  bowire-lockup-dark.svg / -light.svg   Circle-B mark + "Bowire" wordmark.
  bowire-mark-dark.svg   / -light.svg   The Circle-B mark alone (tight spaces).

  bowire-brand-guidelines.pdf           One-page brand sheet (overview).

The full, detailed brand guide (clear-space construction, minimum sizes, colour
tokens, co-branding, file formats, accessibility, usage rights) lives at:
  https://bowire.io/docs/brand-guide.html

The Bowire software is open source (Apache-2.0). The Bowire name and logos are
trademarks of Kuestenlogik; please follow the guidelines and don't alter the
marks. Questions: info@kuestenlogik.com

Overview & downloads: https://bowire.io/brand.html
